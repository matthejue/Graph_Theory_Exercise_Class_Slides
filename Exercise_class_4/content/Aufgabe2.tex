\begin{frame}{Aufgabe \thesection}{Hungarian Algorithm}
    \begin{solutionnoinc}
      \begin{itemize}
        \only<1>{\item[6.] Determine the maximum weighted matching by summing the values of the adjancy matrix that correspond to the edges in the perfect matching of the equality subgraph}
        \only<2>{\item[6.] Determine the maximum weighted matching and minimal weighted vortex cover}
        \begin{itemize}
        \only<1>{
          \item The perfect matching corresponds to choosing $n$ zeros while ensuring that each row and column of the excess matrix only contains one choosen zero
          \item To obtain the maximum weighted matching, one may also sum the values of the adjacency matrix that are in the same positions as the choosen zeros from the excess matrix
        \item \alert{Note:} One may find more than one way of choosing $n$ zeros in the excess matrix. That is ok, because all the choices will have the same total weight}
        \only<2>{
          \item if in the algorithm the equality subgraph reaches a perfect matching $M$ then because of the duality also a maximal weighted matching is found: $\sum_{u_iv_j\in M} w_{i,j}$ and a minimal weighted vertex cover: $\sum_{i} u_i + \sum_{j} v_j$ and both have the same weight
          \item \alert{Optimal assignment:} $\displaystyle \sum_{u_iv_j\in M} w_{i,j} = 4 + 2 + 4 + 3 + 1 = 14 = 3 + 2 + 3 + 3 + 1 + 1 + 1 = \sum_{i} u_i + \sum_{j} v_j$}
        \end{itemize}

      \end{itemize}
